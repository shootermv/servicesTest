Basics

All restApi definitions stored in services.xml file inside ./ directory.
Each service represented by resource: node: 

    
		
			
				
			
			Generates XML output

Path:

this is Request url of service :

Method node:

this is there method of request specified.

action can be 'GET','POST','PUT','DELETE'
id is service name and will be displayed in services catalog
datatype is the output of service - can be json (default) and text

Service Params:

service can send various params:
name is param name
default is param default value

By default params been sent inside if request body (or querystring in case of action "GET")
