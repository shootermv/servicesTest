# servicesTest — the BackboneJS app for inspecting restAPI services
1. displays list of services from XML file
2. displays parmeters, properties and description for each service
3. makes user to be able to launch the ajax request of each service and view the data recieved from server
4. makes user to be able to launch all the ajax requests at once and get indication about the request status
### The frameworks used:
1. BackBone  
2. JqueryMobile
